# Templates

Place your Obsidian note templates here.

The skill uses a decision tree to select the right template based on paper type:

| Template | Use Case |
|----------|----------|
| Template 1 | Fe-based material synthesis + adsorption (full chain) |
| Template 2 | Adsorption only |
| Template 3 | Mesoporous material synthesis only |
| Template 4 | Basic reference |
| Template 5 | Fe-MCM-41 synthesis parameters |
| Template 6 | General literature note |

Create .md template files in this directory and update the template paths
in SKILL.md to match your setup.
